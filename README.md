# Dashboard Multifuncional

Este **Dashboard multifuncional** combina varias herramientas útiles en una sola interfaz minimalista y dinámica. Permite centralizar información clave y optimizar la productividad diaria de manera sencilla y visual.  

Actualmente incluye integración con APIs externas y funcionalidades en desarrollo que ampliarán aún más sus capacidades.  

---

## Funcionalidades

- **Clima en tiempo real:** Consulta el estado del clima y previsiones directamente desde el panel.  
- **Notas rápidas:** Añade, edita y organiza notas para mantener tus ideas y tareas al día.  
- **Marcadores personalizados:** Accede rápidamente a tus páginas favoritas y recursos frecuentes.  
- **Interfaz minimalista:** Diseño limpio, intuitivo y fácil de usar.  
- **Integración futura con Home Assistant:** Automatiza tareas del hogar y controla dispositivos desde el dashboard.  
- **Funciones de IA en desarrollo:**  
  - Generación de recordatorios inteligentes  
  - Sugerencias personalizadas  
  - Respuestas a consultas directamente desde el panel  

---

## Tecnologías utilizadas

- HTML, CSS y JavaScript  
- APIs externas para clima y datos dinámicos  
- Integración futura con Home Assistant y servicios de IA  

---

## Cómo usarlo

1. Abre el dashboard en tu navegador.  
2. Explora las secciones de clima, notas y marcadores.  
3. Añade tus notas y marcadores personalizados según tus necesidades.  
4. Próximamente, conecta tus dispositivos y tareas automatizadas con Home Assistant.
